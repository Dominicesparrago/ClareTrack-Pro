# ClareTrack Pro — Setup Guide

---

## 1. Install Git

Download and install from **https://git-scm.com/downloads**

---

## 2. Install Node.js

Download the **LTS** version from **https://nodejs.org** and install it.

---

## 3. Clone the Project

```bash
git clone https://github.com/Dominicesparrago/ClareTrack-Pro.git
cd ClareTrack-Pro
```

---


## 5. Start the App

```bash
npm run dev
```

Open your browser and go to:

```
http://localhost:3000
```

## Open on Other Devices

1. Run `ipconfig` in the terminal
2. Find your **IPv4 Address** (e.g. `192.168.1.10`)
3. Make sure both devices are on the **same Wi-Fi**
4. Open on the other device:

```
http://192.168.x.x:3000
```
